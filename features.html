Front-End Changes (HTML, CSS, JavaScript)

Calendar View ✅ Working

A dynamic calendar generated with JavaScript, allowing users to view and navigate months.

Add New Event Button ✅ Working

Button opens a popup form where users can add a new event (title, date, time, people).

Edit Event Feature ✅ Working

Edit button appears on each event allowing users to update event details through the form.

Dark Mode Support ✅ Working

Detects dark mode using localStorage and updates the page styling accordingly.

Multiple Themes ✅ Working

Theme options are added, polished, and integrated across the main pages.

Back-End Changes (PHP, MySQL, Authentication)

Event Storage in MySQL ✅ Working

Events are saved to a MySQL database so they persist and show up for all users.

Fetch Events via PHP API ✅ Working

JavaScript fetches event data from the server through PHP endpoints and displays them live.

User Authentication ✅ Working

Login system ensures users are authenticated before accessing certain features like adding events.

Event Ownership (Per User) ✅ Working

Events are specific to users so each user can create and manage their own events.
