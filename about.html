Responsibilites/Roles

We have a set division of labor established, but are more than capable to switch and take over various parts of this project. For the most part, the following is what was accomplished by each person:

  • Dev Patel:

    I implemented the login and registration services, as well as the password recovery functionality. I made sure that proper
    error handling was incorporated along with setting up the database and tables to store important information. I also created
    user profiles for each registered user, where they can upload a profile picture, write a bio, and edit their display name.
    I also allowed authenticated users the option to search up others in order to view their profiles. Finally, I implemented some
    styling for all of the main pages.

  • Drew Goessel: [Calendar events, UI styling, Calendar themes, Dark mode functionality]

    I developed the calendar events features, where authenticated users can create events, modify events, and delete events.
    There is a section on the calendar page that displays all of the upcoming events and puts markers on the grid to showcase these.
    I also created some customization options for users, where they can apply themes to the pages, such as dark mode. I styled
    some of the calendar page specifically.

Interesting Notes

We hope to make a pretty cool and interactive calendar web application and learn throughout this process.
